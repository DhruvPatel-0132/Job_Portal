## ✨ Overview

**TalentForge AI** is a next-generation job portal that leverages artificial intelligence to connect job seekers with recruiters more efficiently.

It focuses on:
- 🎯 Smart job matching  
- ⚡ Faster hiring  
- 🤖 AI-driven automation  
- 📊 Data-driven insights  

---

## 🚀 Features

### 👤 Job Seekers
- AI-powered job recommendations  
- Resume analysis & optimization  
- One-click applications  
- Skill gap insights  
- Application tracking  

### 🏢 Recruiters
- AI candidate matching  
- Smart job posting  
- Resume screening automation  
- Candidate ranking  
- Hiring analytics  

---

## 🧠 AI Capabilities

- 🔍 Resume Parsing  
- 🤖 Job-Candidate Matching

🔐 Authentication Flow
----------------------

### Multi-Step Registration

Step 1 → Email / Phone + Password\
Step 2 → First Name + Last Name\
Step 3 → Role Selection (Job Seeker / Recruiter)

* * * * *

🔄 User Flow
------------

### 👤 Job Seeker

1.  Register / Login
2.  Complete AI onboarding
3.  Get job recommendations
4.  Apply to jobs
5.  Track applications

### 🏢 Recruiter

1.  Register / Login
2.  Select recruiter type
3.  Post jobs
4.  AI suggests candidates
5.  Manage hiring pipeline
